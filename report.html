Development Lifecycle

  1. Backend Initialization: Setup Express server, connect to MongoDB, define Schema (User, Restaurant, Order).
  2. API Development: Created secure endpoints (POST /auth, GET /restaurants) protected by JWT middleware.
  3. Web Integration: Built Next.js pages consuming APIs via Axios. Implemented Firebase Login.
  4. Mobile Development: Created Flutter UI with Provider state management. Integrated `google_sign_in` bridged to Firebase.
  5. Optimization: Added `flutter_launcher_icons`, optimized build size, fixed authorized domains.
  6. Deployment: Deployed Backend/Web to Render. Generated Signed APK for Android.
